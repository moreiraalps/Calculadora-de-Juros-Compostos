# Calculadora-de-Juros-Compostos

**HTML, CSS & JS**

Uma aplicação web interativa desenvolvida para simular a evolução de investimentos baseada em juros compostos. O projeto foca em precisão matemática e uma interface de usuário (UI) limpa e funcional.

## 🎯 Objetivo do Projeto
Demonstrar a aplicação de **JavaScript Vanilla** na resolução de problemas financeiros, manipulação do DOM e tratamento de dados de entrada (inputs) para exibição em tempo real.

## 🧮 Lógica Matemática
O sistema aplica a fórmula clássica de juros compostos:

$$M = P \cdot (1 + i)^n$$

Onde:
- **M**: Montante final (Total acumulado)
- **P**: Capital inicial (Aporte)
- **i**: Taxa de juros mensal (convertida de percentual para decimal)
- **n**: Período de tempo em meses

## 🚀 Funcionalidades
- **Cálculo Automático:** Processamento instantâneo ao clicar no botão.
- **Formatação de Moeda:** Utiliza a API `Intl.NumberFormat` para exibir os resultados no padrão monetário brasileiro (R$).
- **Diferenciação de Valores:** Exibe o total bruto e, separadamente, o valor acumulado apenas em juros.
- **Design Responsivo:** Interface adaptável para diferentes tamanhos de tela.

## 🛠️ Como Visualizar
Como o projeto utiliza apenas tecnologias front-end nativas, não é necessária instalação:
1. Faça o download do arquivo `index.html`.
2. Abra o arquivo em qualquer navegador moderno (Chrome, Edge, Firefox).
